## Command list

start - Start Alfred
hello - Say hi to Alfred
cancel - Exit any conversation you have with Alfred
getdinner - Get tonight's dinner details
startdinner - Start tonight's dinner attendance
joindinner - Join tonight's dinner
leavedinner - Leave tonight's dinner
enddinner - Cancel tonight's dinner
getwifipassword - Get the WIFI password
setwifipassword - Ask Alfred to remember the WIFI password
removewifipassword - Ask Alfred to forget the WIFI password
getcdcvouchers - Get the link for CDC Vouchers
setcdcvoucherlink - Ask Alfred to remember the link for CDC Vouchers
removecdcvoucherlink - Ask Alfred to forget the link for CDC Vouchers
